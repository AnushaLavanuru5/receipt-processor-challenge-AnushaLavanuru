## **How to Run**
 
### **Run with Go:**
```bash
# Build and run the application
go run ./main.go
```
 
### **Run with Docker:**
```bash
# Build the Docker image
docker build -t receipt-processor .
 
# Run the Docker container
docker run -p 8080:8080 receipt-processor
```
 
Now your API is accessible at **http://localhost:8080**.
